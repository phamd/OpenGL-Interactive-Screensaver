  Key    | Function
--------------------------------
   h     | Show/Hide Help Menu!
   1     | Dot Mode 
   2     | Line Mode 
   3     | Polygon Mode 
 <space> | Finish polygon 
   +     | Increase speed 
   -     | Decrease speed 
   t     | Increase vertex size 
   g     | Decrease vertex size 
   r     | Reset scene 
   a     | Randomize scene 
   f     | Toggle fullscreen
   q     | Quit program
